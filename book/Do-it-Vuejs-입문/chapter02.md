### 2. Hello Vue.js! 프로젝트 만들기

#### 뷰 시작하기

* HTML 파일 생성 → 뷰 소스 코드 추가 → 브라우저에서 실행

```HTML
<div id="app">{{ message }}</div>
<script src="https://cnd/jsdelivr.net/npm/vue@2.5.2/dist/vue.js"></script>
<script>
    new Vue({
        el: '#app',
        data: {
            message: 'Hello Vue.js!'
        }
    })
</script>

```
